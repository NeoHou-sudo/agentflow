# ✏️ E-commerce Listing Optimization Workflow / Listing 优化工作流

> **适用场景**: 新品上架或优化现有 listing  
> **难度**: ⭐⭐

---

## 📝 Exact Prompt / 完整提示词

### 中文版

```
你是一位亚马逊 listing 优化专家。请帮我优化以下产品的亚马逊 listing。

## 产品信息
- 产品名称：[产品名]
- 产品特点：[特点1、特点2、特点3]
- 目标客户：[学生/商务/游戏等]
- 差异化亮点：[与竞品相比的优势]

## 需要优化的部分
[ ] 标题优化
[ ] 五点描述（Bullet Points）
[ ] 产品描述（Description）
[ ] 搜索关键词（Backend Keywords）

## 目标市场
Amazon.ca 加拿大站

## 现有 listing（如有）
[粘贴现有标题/描述]

## 要求
1. 标题包含核心关键词，控制在 200 字符内
2. 五点描述突出产品核心卖点，痛点+解决方案
3. 产品描述讲述产品故事，建立情感连接
4. 搜索关键词覆盖长尾词

请直接给出优化后的内容。
```

### English Version

```
You are an Amazon listing optimization expert. Help me optimize the following product listing for Amazon.ca.

## Product Info
- Product name: [name]
- Key features: [f1, f2, f3]
- Target customer: [Student/Business/Gaming]
- Differentiation: [competitive advantage]

## Sections to optimize
[ ] Title
[ ] Bullet Points
[ ] Product Description
[ ] Backend Keywords

## Requirements
1. Title: Include core keywords, within 200 characters
2. Bullets: Highlight core selling points, pain point + solution
3. Description: Tell product story, build emotional connection
4. Keywords: Cover long-tail search terms

Provide optimized content directly.
```
