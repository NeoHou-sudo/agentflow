# 🔬 Differential Diagnosis Support Workflow / 鉴别诊断辅助工作流

> **适用场景**: 辅助医生或患者进行鉴别诊断思考  
> **来源**: 循证医学顾问 Agent  
> **难度**: ⭐⭐⭐⭐

---

## ⚠️ 免责声明

> **本工作流仅用于学习参考，诊断决策必须由执业医师做出。**  
> **This workflow is for educational reference only; diagnosis must be made by licensed physicians.**

---

## 📝 Exact Prompt / 完整提示词

### 中文版

```
你是一位临床诊断专家。请帮我进行鉴别诊断分析。

## 患者信息
- 年龄/性别：[如：45岁男性]
- 主诉：[主要症状]
- 伴随症状：[其他症状]
- 病程：[如：突发/逐渐加重/间歇性]
- 既往史：[相关病史]
- 用药史：[相关用药]
- 过敏史：[药物过敏]

## 已有的检查结果（如有）
- 实验室检查：[数值]
- 影像学：[描述]
- 其他：[心电图等]

## 分析要求

### 1. 主要鉴别诊断列表
列出可能的诊断，按可能性排序：
1. [最可能]
2. [次可能]
3. [需排除]

### 2. 支持点与不支持点
对每个诊断分析：
- 支持该诊断的证据
- 不支持该诊断的证据

### 3. 建议的进一步检查
- 病史补充询问
- 体格检查重点
- 实验室检查建议
- 影像学检查建议

### 4. 紧迫性评估
- 是否需要紧急处理
- 随访时间节点

### 5. 参考文献
- 相关指南
- 典型研究

请给出结构化分析，保持客观。
```
