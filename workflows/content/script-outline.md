# 🎬 Content Script Outline Workflow / 分镜脚本生成工作流

> **适用场景**: AI 短剧/短视频脚本创作  
> **来源**: AI 短剧编剧 Agent  
> **难度**: ⭐⭐⭐

---

## 📝 Exact Prompt / 完整提示词

### 中文版

```
你是一位二次元 AI 短剧编剧。请帮我创作一个短视频分镜脚本。

## 基本信息
- 作品类型：[甜宠/职场/悬疑/搞笑/其他]
- 平台：[抖音/快手/B站/小红书]
- 时长：[30秒/1分钟/3分钟]
- 目标受众：[年轻女性/学生/职场人]

## 故事概要
[一句话概括核心剧情]

## 角色设定
- 女主：[性格、身份、外貌特点]
- 男主：[性格、身份、外貌特点]
- 配角（如有）：[简要描述]

## 需要的内容

### 1. 分镜脚本（按场景分）
每个场景包含：
- 场景编号/场景描述
- 画面/镜头描述
- 台词/对白
- BGM/音效提示

### 2. AI 生图提示词
每个关键场景提供：
- 人物描述（服装、表情、姿势）
- 环境描述
- 画面风格
- 完整版 Stable Diffusion / Midjourney 提示词

### 3. 封面建议
- 封面图片描述
- 标题建议

### 4. 发布策略
- 发布时间建议
- 标签建议
- 互动引导话术

请给出完整内容。
```

### English Version

```
You are an AI short drama scriptwriter. Help me create a short video script.

## Basic Info
- Genre: [Romance/Career/Suspense/Comedy/Other]
- Platform: [TikTok/Kuaishou/Bilibili/Xiaohongshu]
- Duration: [30s/1min/3min]
- Target audience: [Young women/Students/Professionals]

## Story Summary
[One sentence core plot]

## Character Setup
- Female lead: [Personality, identity, appearance]
- Male lead: [Personality, identity, appearance]

## Deliverables

1. **Scene-by-scene Script**
   - Scene # / Scene description
   - Visual/camera description
   - Dialogue
   - BGM/SFX cues

2. **AI Image Prompts**
   - Character description
   - Environment description
   - Style
   - Full SD/MJ prompt

3. **Cover Design**
   - Cover image description
   - Title suggestions

4. **Post Strategy**
   - Best posting time
   - Hashtags
   - Engagement CTA

Provide complete content.
```
