# 🎬 短视频脚本工作流

> **适用场景**: 想做抖音/视频号/小红书视频，不知道怎么写脚本  
> **来源**: 短视频平台爆款规律  
> **效果**: 输入主题 → 2分钟出完整脚本（开场钩子+正文+行动号召）  
> **难度**: ⭐（复制粘贴即可）

---

## 一句话

输入视频主题 → AI 生成完整短视频脚本，包含开场钩子、正文结构、结尾引导。

---

## 提示词

```
你是一位抖音/视频号/小红书短视频资深编导。

请根据我给的主题，生成一个完整的短视频脚本。

## 我的主题
[你的视频主题，例如：分享一个副业赚钱思路 / 测评一款产品 / 教一个生活小技巧]

## 视频信息
- 平台：[抖音 / 视频号 / 小红书]
- 时长：[30秒 / 60秒 / 90秒]
- 目标观众：[大学生 / 打工人 / 宝妈 / 创业小白 等]

## 请生成完整脚本，包含：

### 1. 开场钩子（前3秒）
- 设计一个强钩子，让用户停留
- 方案：反常识 / 数字冲击 / 直接提问 / 冲突场景

### 2. 正文结构
- 分3-4个段落
- 每段有明确的信息点
- 使用"因为...所以..."、"不是...而是..."等逻辑连接

### 3. 视觉提示
- [画面：...]
- [字幕：...]
- [音效：...]

### 4. 结尾引导（CTA）
- 引导评论 / 引导关注 / 引导分享
- 设计一个互动钩子

## 输出格式

```
【开场钩子】
[画面/字幕/口播内容，3秒]

【正文第一段】
[时间戳 | 内容 | 视觉提示]

【正文第二段】
...

【结尾CTA】
[具体引导语]
```

---

## 脚本公式库（直接套用）

### 公式1：痛点+方案（最通用）
```
开场："你是不是也这样？[描述痛点]"
正文："其实很简单，只需要[解决方案]"
      "第一步...[第二步...]"
结尾："学会了就关注我，每天分享干货"
```

### 公式2：反常识+揭秘
```
开场："所有人都搞错了！[普遍误解]"
正文："真相是[揭示真相]"
      "这背后是因为[解释原理]"
结尾："想知道更多？评论区告诉我"
```

### 公式3：结果前置+过程
```
开场："我用了这个方法，[具体结果]"
正文："第一步...[详细步骤]"
      "第二步..."
      "第三步..."
结尾："收藏起来慢慢试，用完记得回来谢我"
```

### 公式4：测评/对比型
```
开场："[产品A] vs [产品B]，到底哪个好？"
正文："先看[维度1]...[结果]"
      "再看[维度2]...[结果]"
结尾："综合来看，我推荐[结论]"
```

## 真实案例

**主题**: 分享一个适合上班族的副业赚钱思路  
**平台**: 抖音  
**时长**: 60秒  
**目标**: 打工人，想找副业但没方向

**AI 生成脚本**:

```
【开场钩子】（0-3秒）
[画面：坐在办公桌前，表情疲惫]
[字幕：月入3万的我，白天工资8000]
[口播：下班后这个动作，让我多赚2万]

【正文第一段】（3-20秒）
[画面：电脑屏幕特写]
[字幕：下班后2小时，我做了什么]
口播：不是什么复杂的副业，就是把工作中积累的[技能]
      做成了一个小服务，在闲鱼上卖
      第一周就出了3单

【正文第二段】（20-40秒）
[画面：订单截图、收入截图]
[字幕：具体怎么定价？]
口播：定价很简单，就看你的时间值多少钱
      我当时定价是[日薪/8]，一单花我1小时
      但后来发现，复用性强的模板可以卖很多份

【正文第三段】（40-55秒）
[画面：产品页面截图]
[字幕：具体操作步骤]
口播：操作步骤分三步
      第一，选一个你工作中最熟练的技能
      第二，把它整理成一份可交付的东西
      第三，去闲鱼/小红书发布，等人来买

【结尾CTA】（55-60秒）
[画面：对着镜头竖大拇指]
[字幕：想看后续操作细节吗？]
[口播：如果这条视频过1000赞，下期出详细教程]
[字幕：关注我，每天一个搞钱思路]
```

## 平台适配指南

| 平台 | 开场策略 | 结尾引导 |
|------|----------|----------|
| 抖音 | 前3秒必须有强钩子，完播率是核心 | 引导评论："你们觉得呢？" |
| 视频号 | 前3秒可以用故事型，情感共鸣重要 | 引导转发："转给需要的朋友" |
| 小红书 | 封面标题党+前3秒美感 | 引导收藏："先收藏，一定会用到" |
