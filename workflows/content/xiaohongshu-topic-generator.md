# 🎬 小红书爆款选题生成器

> **适用场景**: 找不到选题、不知道发什么、发的内容没流量  
> **来源**: 小红书热门内容规律 + 真实爆款拆解  
> **效果**: 3 分钟生成 15 个带爆火理由的选题  
> **难度**: ⭐（复制粘贴即可）

---

## 一句话

输入你的行业/赛道 → AI 生成 15 个近期可能引爆流量的选题，每个都带爆火理由。

---

## 提示词

```
你是一位小红书资深运营，对平台热点和用户真实需求有敏锐洞察。

请根据我的赛道/行业，生成15个近期可能引爆流量的选题。

## 我的赛道
[你的赛道，例如：美食测评 / 职场穿搭 / 母婴 / 健身 / 副业]

## 生成要求

每个选题必须包含：
1. **选题标题**（20字以内，有吸引力）
2. **爆火理由**（一句话说明为什么它能火，吸引什么人，满足什么需求）
3. **切入角度**（具体不宽泛，比如"肿眼泡如何画眼妆"而不是"如何化妆"）

## 融合以下元素
- 当下季节 / 节日 / 热门影视剧 / 流行梗
- 覆盖不同用户痛点：避坑、省钱、效率、变美、搞钱、情感共鸣
- 融入近期小红书热搜词（如"听劝"、"显眼包"、"牛马"等）

## 输出格式

| # | 选题 | 爆火理由 | 切入角度 |
|---|------|----------|----------|
| 1 | 听劝！这5款"马年爆火面包"到底是神颜还是诈骗？ | 结合"听劝"热搜 + 马年热度，解决选择困难 | 价格/口味/造型"抽象程度"三维测评 |
| ... | ... | ... | ... |
```

---

## 变体：根据不同阶段生成

### 冷启动阶段（0-1000粉）
```
我刚做小红书 [你的赛道]，粉丝不到1000，请生成适合冷启动期的选题。
要求：容易引发互动（评论/收藏）、门槛低、普通人也能参与。
```

### 增长阶段（1000-10000粉）
```
我的小红书 [你的赛道] 已经有5000粉丝，需要提升内容质量和影响力。
请生成能吸引精准用户、建立个人IP的选题。
要求：有干货价值、能展示专业度、容易引发转发。
```

### 变现阶段（10000粉+）
```
我的小红书 [你的赛道] 已经有1万粉丝，想开始接广告/带货。
请生成既能留粉又能促进转化的选题。
要求：软性植入空间大、用户购买意愿强。
```

## 使用技巧

1. **每周跑一次**：小红书热点变化快，每周更新一次选题库
2. **结合节日**：节假日前1-2周植入节日元素，流量翻倍
3. **看评论区**：选一个有潜力的选题后，去小红书搜类似内容，看评论区用户在问什么，围绕那个"未满足的需求"做内容
