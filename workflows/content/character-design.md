# 👤 Character Design Workflow / 角色人设设计工作流

> **适用场景**: 创作短视频/短剧中的角色人设  
> **难度**: ⭐⭐

---

## 📝 Exact Prompt / 完整提示词

### 中文版

```
你是一位内容创作者。请帮我设计一个短视频/短剧角色的人设。

## 基本信息
- 作品类型：[甜宠/职场/悬疑/搞笑]
- 平台：[抖音/快手/B站]
- 角色定位：[男主/女主/反派/配角]

## 角色需求
- 核心性格：[1-2个关键词]
- 身份职业：[如：霸总/学霸/创业者/医生]
- 年龄段：[20-30岁]
- 外貌特征：[如：冷峻帅气/甜美可爱/阳光活力]

## 需要的设计内容

### 1. 基础人设卡
- 姓名（中文+英文名）
- 年龄、职业、外表
- 性格关键词（3-5个）
- 口头禅/标志性动作
- 背景故事（1-2句话）

### 2. 性格详解
- 优势面
- 弱点/BUG
- 行为模式
- 与其他角色的化学反应

### 3. AI 生图提示词
- 基础形象：穿着、职业、表情
- 场景变体：不同场景的穿搭描述
- 完整 SD/MJ 提示词

### 4. 台词风格
- 常用句式示例（5条）
- 说话语气特点

### 5. 与其他角色的关系
- CP 互动模式（如适用）
- 与配角的关系

请给出完整角色设计。
```
