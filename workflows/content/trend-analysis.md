# 📊 Content Trend Analysis Workflow / 内容趋势分析工作流

> **适用场景**: 分析短视频平台的内容趋势，找到爆款选题  
> **难度**: ⭐⭐

---

## 📝 Exact Prompt / 完整提示词

### 中文版

```
你是一位短视频内容策略师。请帮我分析 [平台] 当前的内容趋势。

## 分析平台
- 平台：[抖音/快手/B站/小红书]
- 目标领域：[如：情感、职场、美食、知识]

## 背景信息
- 我的账号定位：[如：职场成长]
- 我擅长的内容类型：[如：干货分享/故事演绎]
- 粉丝画像：[如：22-30岁职场女性]

## 需要分析的内容

### 1. 近期爆款话题
- 当前流量最高的 10 个话题/挑战
- 热度上升趋势（新兴/平稳/衰退）

### 2. 热门内容形式
- 近期火的视频结构/节奏
- 常见的开场方式
- 互动形式（评论区引导等）

### 3. 竞品分析
- 同领域 TOP 10 账号最近在做什么
- 他们的爆款规律

### 4. 差异化机会
- 目前内容中未被满足的需求
- 可以切入的细分角度

### 5. 可执行建议
- 近期可以追的 3 个热点
- 内容选题建议
- 发布节奏建议

请给出结构化分析。
```
