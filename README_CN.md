# AgentFlow

> 18+ 真实 AI Agent 工作流。拿来即用，2 分钟出结果。

---

## 一句话定位

不是理论，不是「10个超好用的ChatGPT提示词」——是真实跑过、真出过结果的工作流。

---

## 实际效果示例

### 电商选品（3分钟）

**你的输入：**
> 品类：笔记本电脑 / 芯片：Intel N100 / 屏幕：15.6寸 IPS  
> 成本：¥800 / 目标：Amazon.com 美国站

**跑完后：**
> ✅ $280–$350 价位只有 3 个主要玩家，存在空位。  
> 建议售价：$299–$329，预估毛利：22–26%。  
> 风险提示：FCC 认证必须，CE 认证覆盖欧盟。  
> 建议先测 20 台小批量。

---

### 持仓诊断（2分钟）

**你的输入：**
> 持仓：电商店铺（¥50,000）+ 成长型基金（¥30,000）  
> 总资产：¥120,000 / 目标（1年）：¥100,000

**跑完后：**
> ⚠️ 电商店铺占总资产 41%，集中度过高。  
> 建议 30 天内调整为 60/40（权益/债券）配置。  
> 当前回撤风险：高。  
> 补充：加入低费率标普 500 ETF，地理分散。

## 工作流长什么样

每个工作流都是一套完整的提示词 + 输出格式说明。

**举个子：选品调研工作流的核心提示词**

```
你是一位跨境电商选品专家。请帮我分析这款产品是否值得在 Amazon 销售。

## 产品信息
- 品类：[笔记本电脑 / 数码配件 / 家居用品]
- 配置：[具体参数]
- 成本：[你的成本价]

## 请分析
1. 市场机会：搜索结果数、月销量估算、竞争密度
2. 竞争分析：TOP10 竞品价格区间、找价格空缺
3. 定价建议：成本 → 建议售价、毛利估算
4. 风险提示：认证、品牌、竞争等坑点
```

**对应的输出格式**

```
## 结论：🟡 可尝试

## 市场机会
| 搜索结果 | 预估月销 | 主要玩家 | 市场评级 |
|----------|----------|----------|----------|
| ~200个   | 300-500台 | 12家   | 🟡 中等  |

## 竞争分析
| 价格区间    | 玩家数 | 机会 |
|-------------|--------|------|
| $180-$220  | 6家    | 🟡   |
| $280-$350  | 3家    | 🟢   |

## 定价建议
- 建议售价：$299–329
- 预估毛利：22–26%
- 建议先测 20 台
```

## 包含的工作流

**电商运营**（3个）
- 选品调研 / 竞品分析 / Listing 优化

**投资分析**（3个）
- 持仓诊断 / 市场分析 / 资产配置

**内容创作**（3个）
- 短视频脚本 / 选题生成 / 内容批量产出

**医学研究**（2个）
- 临床决策 / 鉴别诊断

**安全审计**（2个）
- 威胁建模 / 漏洞扫描

## 怎么用

1. 打开 `workflows/` 找到你的场景
2. 复制提示词，粘贴到任意 AI Agent
3. 改改参数，执行

每个工作流都有完整提示词（中文 + 英文）+ 输出格式 + 真实案例。

**Q: 需要技术背景吗？**
