## 🎯 这是什么？

AgentFlow 是一个**经过 500+ 小时真实生产使用验证的 15+ 套 AI Agent 工作流**开源库，覆盖 5 大领域。

不是理论，不是泛泛的提示词列表——每一套都是**可直接复制使用的工作流**。

## 🚀 快速开始

### 第一步：选择工作流
浏览 `workflows/` 目录，找到你的使用场景。

### 第二步：复制提示词
将提示词模板粘贴到你的 AI Agent（ChatGPT、Claude、QClaw 等）。

### 第三步：运行并获得结果
执行工作流，获得可直接使用的结果。
