## 🚀 Quick Start / 快速开始

### Step 1: Pick a workflow / 选一个工作流
Browse the `workflows/` directory and find your use case.

### Step 2: Copy the prompt / 复制提示词
Copy the exact prompt template into your AI Agent (ChatGPT, Claude, QClaw, etc.).

### Step 3: Run & get results / 运行并获得结果
Execute the workflow and get production-ready output.

```markdown
# Example / 示例
1. Open your AI Agent
2. Paste the prompt from workflows/ecommerce/product-research.md
3. Enter your product keywords
4. Get a complete market analysis report
```
