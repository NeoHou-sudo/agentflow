🦞 部署 OpenClaw 小龙虾
推荐新手 · Windows/macOS/Linux · 免费开源
1Windows 一键安装（30秒）
下载安装包后，右键解压到纯英文路径（不含中文/空格），找到「Openclaw Windows 一键启动.exe」双击即可。
2macOS / Linux 安装
打开终端，执行一键安装命令：
curl -fsSL https://openclaw.ai/install.sh | sh
3获取 API Token（免费渠道）
推荐：DeepSeek Token（每日免费额度）
1. 访问 https://chat.deepseek.com 注册
2. 点击左下角「API Keys」→ 创建新密钥
3. 复制密钥，填入 OpenClaw 配置页
4导入工作流
将本仓库 workflows/ 目录下的工作流文件内容复制到 OpenClaw Agent 的对话中即可使用。
☁️ 云端一键部署（无需本地）
百度智能云 / 阿里云：搜索「OpenClaw 镜像」→ 一键购买部署 → 最快 3 分钟上线
参考：https://openclaw.ai/install
⚡ 部署到 Coze 扣子
云端运行 · 无需服务器 · 5分钟上线
1注册 Coze 账号
访问 https://www.coze.cn/home，用抖音/手机号登录，进入工作台。
2创建工作流
点击「创建工作流」→「从空白开始」→ 命名（如"小红书文案批量生成"）。
3配置节点
开始节点：添加变量 topic（字符串）

大模型节点：模型选 Claude 或豆包，Prompt 复制本仓库对应工作流的提示词

结束节点：输出大模型结果
4获取 API Token
1. Coze API → 授权 → 个人访问令牌
2. 点击「添加」→ 填写名称 → 设置过期时间 → 勾选权限
3. 复制并保存令牌（只显示一次）
4. 令牌格式：Bearer pat_xxxxxxxxxxxx
5发布 Bot
点击右上角「发布」→「发布为 Bot」→ 设置名称 → 完成。

🔧 部署 Dify
开源免费 · 支持私有化 · Docker 一键启动
1Docker 一键启动（推荐）
克隆仓库并启动（访问 http://localhost:80 注册账号）：
git clone https://github.com/langgenius/dify.git
cd dify/docker
cp .env.example .env
docker compose up -d
2导入工作流模板
Dify 支持 DSL 文件导入：
1. 克隆本仓库：git clone https://github.com/NeoHou-sudo/agentflow
2. 在 Dify 工作台点击「创建应用」→「导入 DSL」
3. 选择本仓库 platforms/dify/ 下的模板文件
3配置 API Key
支持多模型：OpenAI / Claude / DeepSeek / 阿里通义

推荐 DeepSeek（成本低，质量好）：
1. https://platform.deepseek.com 注册
2. API Keys → 创建密钥
3. 填入 Dify 的「模型供应商」设置
4华为云一键部署（免配置）
登录 support.huaweicloud.com → 搜索「Dify」→ 选择「快速部署」→ 一键创建资源栈。
